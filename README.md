# Sistema de Gestión de Recursos Humanos (Full Stack)

## Descripción

Este es un proyecto **Full Stack** diseñado para gestionar la nómina de empleados de una organización. La aplicación permite realizar operaciones CRUD (Crear, Leer, Actualizar y Eliminar) completas, conectando un frontend moderno en **React** con un backend robusto en **Spring Boot** y persistencia en **MySQL**.

El objetivo de este proyecto es demostrar la implementación de una arquitectura en capas, buenas prácticas de desarrollo y la integración entre tecnologías líderes en la industria.

## Tecnologías Utilizadas

### Backend (API REST)
* **Lenguaje:** Java 21 (JDK 21)
* **Framework:** Spring Boot 4.0.0
* **Datos:** Spring Data JPA & Hibernate
* **Build Tool:** Maven

### Frontend (Cliente Web)
* **Librería:** React JS
* **Navegación:** React Router Dom
* **Cliente HTTP:** Axios
* **Estilos:** CSS3 / Bootstrap (Diseño responsivo)

### Base de Datos
* **Motor:** MySQL
* **Configuración:** `update` automático de DDL.
